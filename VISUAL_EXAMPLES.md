## 🎬 **Demo Flow Visualization**

### **Step 1: Installation** (30 seconds)
```bash
git clone https://github.com/xk0dex/empathy.git
cd empathy
pip install -r requirements.txt
```

### **Step 2: Basic Demo** (60 seconds)
```bash
python3 demo.py
# Shows complete analysis with real metrics
```

### **Step 3: Web Dashboard** (90 seconds)
```bash
python3 demo.py --web
# Interactive dashboard at http://localhost:8080
```
